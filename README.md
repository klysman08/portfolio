## **Project structure**
```
public/
└── svg/
src/
├── assets/
├── components/
├── layouts/
└── pages/
```

## **Local configuration** 
1. Clone the repo:  
```
git clone https://github.com/garaven/portfolio
```
2. Install dependencies:
```  
npm install
```
3. Start the development server:
```  
npm run dev
```
